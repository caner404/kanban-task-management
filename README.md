## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

- Clone this repo:

```sh
git clone https://github.com/caner404/kanban-task-management.git
```

- Install dependencies:

```sh
npm install
```

- Build command:

```sh
npm run build
```

- Live server:

```sh
npm run dev
```

- Open storyboook:

```sh
npm run storybook
```

- Cypress:

```sh
npm run dev
npm run cy:open
```
